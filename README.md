# Re-hosting of Drew DeVault's Wayland book

This repository is a re-hosting of Drew DeVault's [Wayland book](https://wayland-book.com/). The original source code can be found [here](https://git.sr.ht/~sircmpwn/wayland-book).

The reason for making this re-hosting is to make the book website more usable for mobile browsers, as it is not possible to view the table of contents or change theme in the original website.

In particular, the only changes made to the book website are:
- Used `mdbook`'s built-in theme and templates instead.
- Removed one deprecated option in `book.toml`.
- Added a note to the Introduction page to credit the author and link to the original website.

Aside from these changes, the actual contents of the book remain exactly the same.
